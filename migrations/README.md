# Миграции для портала "Ангар"

## Применение миграций

Вы можете выполнить SQL-скрипты по порядку в вашей базе данных MySQL:

1. `001_create_tables.sql` – создание всех таблиц.
2. `002_seed_data.sql` – добавление ролей и тестового администратора (пароль admin123).

Либо использовать автоматический скрипт `migrate.php` в корне проекта, который выполнит оба файла.

## Примечания

- Для корректной работы хеша пароля убедитесь, что ваш PHP использует стандартный bcrypt.
- При необходимости отредактируйте email и пароль администратора в `002_seed_data.sql` или создайте администратора вручную.
- Для production рекомендуется удалить тестового администратора и создать своего через регистрацию или отдельный скрипт.
